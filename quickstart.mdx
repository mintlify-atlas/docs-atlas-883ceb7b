## Prerequisites

Before you start, make sure you have the following installed:

- [Node.js](https://nodejs.org/en/download/) 18 or later
- [pnpm](https://pnpm.io/installation) v9 or later
- [Git](https://git-scm.com/downloads)

## Set up locally

<Steps>
  <Step title="Clone the repository">
    ```bash
    git clone https://github.com/skyhancloud/skyhan.cloud.git
    ```
  </Step>

  <Step title="Navigate into the project">
    ```bash
    cd skyhan.cloud
    ```
  </Step>

  <Step title="Install dependencies">
    ```bash
    pnpm install
    ```
  </Step>

  <Step title="Configure environment variables">
    Copy the example env file and fill in your values:

    ```bash
    cp .env.example .env
    ```

    The `.env` file contains the following variables:

    ```env .env
    RYBBIT_ANALYTICS_SITE_ID=
    RYBBIT_ANALYTICS_URL=
    ```

    <Note>
      Both analytics variables are optional. Leave them empty to disable analytics entirely. See [Analytics](/configuration/analytics) for setup instructions.
    </Note>
  </Step>

  <Step title="Start the development server">
    ```bash
    pnpm dev
    ```

    The dev server uses [Turbopack](https://turbo.build/pack) for fast builds. Open [http://localhost:3000](http://localhost:3000) in your browser — you should see the portfolio homepage.
  </Step>
</Steps>

## Available scripts

| Script | Command | Description |
|--------|---------|-------------|
| Dev server | `pnpm dev` | Starts Next.js with Turbopack |
| Build | `pnpm build` | Creates a production build |
| Start | `pnpm start` | Runs the production build |
| Lint | `pnpm lint` | Runs ESLint |

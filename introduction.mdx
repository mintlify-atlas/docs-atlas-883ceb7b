# skyhan.cloud

skyhan.cloud is the personal portfolio and blog of **Gökhan Bulut**, a developer from Turkey. The site is open-source and built on modern web technologies, designed with a minimal, warm aesthetic.

## Tech stack

| Technology | Purpose |
|---|---|
| [Next.js 15](https://nextjs.org) | React framework with App Router |
| [Tailwind CSS](https://tailwindcss.com) | Utility-first styling |
| [MDX](https://mdxjs.com) | Blog posts with embedded components |
| [Radix UI](https://radix-ui.com) | Accessible UI primitives (Tooltip) |
| [Zustand](https://zustand-demo.pmnd.rs) | Lightweight client state management |
| [Zod](https://zod.dev) | Environment variable validation |
| [lantern.rest](https://lantern.rest) | Live Discord presence API |
| [Rybbit](https://www.rybbit.io) | Privacy-friendly analytics |

## Project structure

```
skyhan.cloud/
├── app/
│   ├── blogs/              # Blog posts as MDX pages
│   │   └── [slug]/
│   │       └── page.mdx
│   ├── layout.tsx          # Root layout with fonts and analytics
│   └── page.tsx            # Homepage
├── components/             # Reusable React components
│   ├── blogs/              # Blog-specific components
│   ├── avatar.tsx
│   ├── discord-status.tsx
│   ├── inline-quote.tsx
│   ├── projects.tsx
│   ├── tooltip.tsx
│   └── works.tsx
├── styles/                 # Global CSS with design tokens
├── utils/                  # Utility functions
├── stores/                 # Zustand state stores
├── types/                  # TypeScript type definitions
└── public/                 # Static assets
```
